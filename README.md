## /continue

Lists 5 latest sessions, you choose, it reads session's transcript, any handoff notes and continues work.

## Install

```bash
git clone https://github.com/eikonoikari1/continue.git ~/.claude/skills/continue
```

## Usage

```
/continue          # pick a recent session to continue
/continue --auto   # skip confirmations
/continue --setup  # reconfigure
```
